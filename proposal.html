Project Title: Kinetic Sisplay

This project would create a machine that could arrange balls periodically along some sort of string. There would be for instance 100 strings in a 10x10 grid hanging vertically the strings could move and coupled with the machines ability to arrange the balls at different lengths the overall effect would be a 3d image. The balls could be attached to the stings via friction but not so much that a claw or some sort of sphincter could change the ball's position. The sting would then be moved via some pulley moving the balls to the right height. This is the coolest kinetic display I found but another idea would be to create a screen using possibly pistons or rotating tiles to create a non light based screen.
