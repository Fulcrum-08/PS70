📌 Project Title: 3D LiDAR Mapper

For my final project, I plan to create a compact 3D LiDAR Mapper — a device that uses a spinning laser distance sensor to scan its surroundings and generate a point cloud map.

The mapper will consist of a microcontroller, a LiDAR sensor, and a rotating mount (possibly stepper motor-driven). Using code, I’ll collect distance measurements while tracking rotation angle to build a 360° spatial map.

Ideally, I’d like to visualize the data in real-time or export it into formats usable by 3D software. This project combines hardware control, real-time programming, and data visualization — and it’s a great way to explore robotics and spatial mapping, two fields I’m very excited about.
